大作业中使用了多光源的渲染场景，使用了环境光，点光源和模拟手电筒发出的聚光灯。并可以通过键盘的WSAD键切换观察位置，可以通过鼠标的移动切换观察角度，可以通过鼠标滚轮切换视距，可以通过方向上下键切换聚光灯的强弱，可以通过方向左右键切换聚光灯颜色。对物体的不同材质设置了不同的参数以实现不同的光照效果，并实现了环境光照，漫反射光照，镜面反射光照，亮度衰减模拟真实的物理特性。通过规律性地变换点光源的位置，增强对冯氏光照模型的模拟的理解。

主要实现功能：
1.WSAD键控制照相机的移动
2.鼠标移动控制视角旋转
3.鼠标滚轮控制视距
4.方向键控制光亮度和颜色
5.实现环境光，点光源，聚光灯光照效果
6.实现漫反射，镜面反射材质
7.实现光亮度衰减，模拟真实效果
8.实现光源移动，在运动中观察光照的变换
9.实现了游戏中自由移动的第一人称视角模拟
10.同一个物体表面的不同部分实现了不同材质，表现不同的光照效果。
